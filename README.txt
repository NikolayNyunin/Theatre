Для запуска веб-приложения необходимо запустить скрипт app.py.
Все формы описаны в файле forms.py,
а вся работа непосредственно с базой данных ведётся через классы в файле db.py.
Для первоначального создания тестовой базы данных
с целью проверки работоспособности сайта можно запустить скрипт test.py.
Чтобы иметь возможность редактировать данные спектаклей и актёров,
нужно залогиниться под данными администратора (логин: admin, пароль: admin123).
При "привязывании" актёров к спектаклю необходимо
указывать id нужных актёров через запятую в соотвествующем поле.
